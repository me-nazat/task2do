# ⚡ Task2Do — The High-Performance Workspace

Task2Do is a premium, AI-native productivity application designed for deep work and seamless organization. Built with a "fluid-first" philosophy, it balances powerful desktop orchestration with an elegant, polished mobile experience.

---

## ✨ Core Pillars

### 📥 Intelligent Inbox
Capture objectives instantly. The smart landing filter identifies critical tasks, surface-level reminders, and upcoming deadlines in a single, high-contrast view.

### 🤖 Generative AI Assistant 
Integrated directly with Gemini and MiMo, our AI chat isn't just a sidekick—it's your project partner. Draft tasks, plan your week, and schedule events via natural language.

### 🏗️ Advanced Workflow Views
*   **Kanban Board**: Visualize your pipeline with horizontal snap-scrolling on mobile and drag-and-drop orchestration on desktop.
*   **Eigenhower Matrix**: Master your priorities with a softened, color-coded grid for Urgent/Important triage.
*   **Premium Calendar**: Manage your time in a high-fidelity scheduling view that adapts perfectly to any screen size.
*   **Habit Engine**: Track your progress with a sophisticated routine tracker.

---

## 📱 Mobile-First Ergonomics

We believe mobile productivity shouldn't feel like a compromise.
*   **Dynamic Bottom Dock**: One-tap access to your primary views.
*   **Floating Action Plus (FAB)**: Rapid entry for new ideas in a thumb-friendly layout.
*   **Glassmorphic Aesthetic**: Sleek, translucent panels and subtle micro-animations that feel premium on every frame.
*   **Touch-Optimized Targets**: Hand-calibrated 44px+ hit areas for frustration-free navigation.

---

## 🛠️ Technical Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Database**: [Turso](https://turso.tech/) (Edge SQLite)
- **Animations**: [Motion](https://motion.dev/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Icons**: [Lucide](https://lucide.dev/)

---

## 🚀 Getting Started

1.  **Clone & Install**
    ```bash
    git clone https://github.com/me-nazat/task2do.git
    cd task2do
    npm install
    ```

2.  **Environment Setup**
    Create a `.env.local` with your database and AI API keys.

3.  **Launch**
    ```bash
    npm run dev
    ```
